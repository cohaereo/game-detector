# Game detector
This crate detects installed games on your system on a per-store basis.

## Supported Stores
| Store           | Windows | Linux | Mac |
|-----------------|---------|-------|-----|
| Battle.net      | ❌       | 〰     | ❌   |
| Epic Games      | ✅       | 〰     | ❌   |
| GOG Galaxy      | ❌       | 〰     | 〰   |
| Microsoft Store | ✅       | 〰     | 〰   |
| Origin          | ❌       | 〰     | ❌   |
| Steam           | ✅       | ❌     | ❌   |

Feel free to contribute unsupported/new stores/platforms
